# 注意事项
1、本机型需要禁用一个XCHI，详细操作参考上方链接，我禁用的是XCHI 1 ，会造成蓝牙不可用。禁用XCHI 0 会造成usb接口和键盘不能用，请谨慎操作！！！ <br>
2、我自己加装了一块 980 1T SSD，原装海力士固态需要禁用，详细操作参考上方链接！ <br>
3、其他bios设置大同小异，安装之前需要设置为混合模式，不然安装后加载核显驱动会自动重启！（bios版本GKEC60WW） <br>
